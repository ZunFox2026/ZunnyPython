Giới thiệu về Media Query

Media Query là một kỹ thuật trong CSS cho phép bạn áp dụng các phong cách khác nhau tùy thuộc vào các điều kiện nhất định, chẳng hạn như kích thước màn hình hoặc thiết bị.

Ứng dụng của Media Query

Media Query được sử dụng rộng rãi trong thiết kế web hiện đại để tạo ra các trang web đáp ứng, tức là có thể thích nghi với các kích thước màn hình và thiết bị khác nhau.
