Tìm hiểu về Media Queries

Media Queries là một tính năng trong CSS cho phép áp dụng các kiểu dáng khác nhau dựa trên các điều kiện cụ thể của thiết bị hiển thị.

Với Media Queries, bạn có thể tạo ra các trang web có giao diện phản hồi linh hoạt trên nhiều loại thiết bị và kích thước màn hình khác nhau.
