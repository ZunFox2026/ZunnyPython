# Giới thiệu Python
## Giới thiệu
Python là một ngôn ngữ lập trình cấp cao, được sáng tạo vào cuối những năm 1980 bởi Guido van Rossum. Python được thiết kế để dễ đọc, dễ viết và dễ học, khiến nó trở thành một lựa chọn phổ biến cho cả người mới bắt đầu và lập trình viên chuyên nghiệp. Python được sử dụng rộng rãi trong nhiều lĩnh vực, bao gồm phát triển web, phân tích dữ liệu, trí tuệ nhân tạo, tự động hóa và nhiều hơn nữa.

## Lý thuyết
Python là một ngôn ngữ lập trình hướng đối tượng, nghĩa là nó tổ chức mã thành các đối tượng và các phương thức hoạt động trên các đối tượng đó. Python cũng là một ngôn ngữ lập trình động, nghĩa là nó không yêu cầu khai báo kiểu dữ liệu trước khi sử dụng. Python có nhiều tính năng như quản lý bộ nhớ tự động, xử lý ngoại lệ và hỗ trợ cho các thư viện và framework đa dạng. Python cũng có một cộng đồng lớn và tích cực, cung cấp nhiều tài nguyên và hỗ trợ cho người học và phát triển.

## Ví dụ
Dưới đây là một số ví dụ cơ bản về Python:
- In ra màn hình: `print("Xin chào, thế giới!")`
- Khai báo biến: `x = 5; y = 10`
- Tính tổng: `tong = x + y; print(tong)`
- Sử dụng vòng lặp: 
  ```
  for i in range(5):
      print(i)
  ```
- Sử dụng hàm: 
  ```
  def chao(name):
      print("Xin chào, " + name)
  chao("Zunny")
  ```

## Bài tập
Để bắt đầu với Python, bạn có thể thực hiện các bài tập sau:
- Viết một chương trình Python để in ra màn hình "Xin chào, thế giới!" 5 lần.
- Tạo một chương trình Python để tính tổng của hai số nguyên.
- Viết một hàm Python để kiểm tra xem một số có phải là số nguyên tố hay không.
- Thực hiện một chương trình Python để in ra các số từ 1 đến 10 bằng cách sử dụng vòng lặp.
Bằng cách hoàn thành các bài tập này, bạn sẽ có được kiến thức cơ bản về Python và sẵn sàng để học sâu hơn về ngôn ngữ lập trình này.
